# Hotel Demand–Supply Recovery Analysis (Japan)

MLIT（国土交通省）「宿泊旅行統計調査」を用いて、  
**コロナ禍前後における宿泊市場の回復度合いを都道府県別に可視化・構造分析**したプロジェクトです。

単純な需要回復の把握に留まらず、  
**供給規模（施設数）・稼働率の安定性（変動リスク）・インバウンド依存度**を組み合わせ、  
市場構造と投資・運営上の示唆を導出することを目的としています。

---

## プロジェクトの目的

本プロジェクトは、  
**「未整備な統計データをETLし、構造的な市場理解と意思決定に耐える示唆を導く」**  
ことを目的とした分析ポートフォリオです。

---

## 結論（Summary）

- 2024年時点で、稼働率の回復が早い県は必ずしも大都市圏に限られず、地方にも多く存在
- 回復度合いは「需要量」だけでなく、  
  **供給規模（施設数）× 稼働率の効率性・安定性**によって構造的に説明可能
- インバウンド依存度が高い市場は高稼働になりやすい一方、**変動リスクも高い傾向**
- 市場規模が比較的小さい地域ほど供給過多になりにくく、  
  **回復を先行して捉えやすい可能性**が示唆される

---

## 主な指標定義

- **`index2019`**  
  - 稼働率の **2019年比回復指数**（年次データは2019年比）
- **`inbound_share`**  
  - 外国人延べ宿泊者数 / 総延べ宿泊者数
- **`demand_supply_ratio`**  
  - 延べ宿泊者数 / 施設数（需給の簡易 proxy）
- **`risk`**  
  - 月次稼働率の変動係数（CV）  
    → 需要変動・季節性リスクの proxy
