🏨 宿泊需要と供給の回復分析 (Hotel Supply and Demand Recovery Analysis)

概要
本リポジトリは、コロナ禍以降の日本の宿泊業における需要（稼働率など）と供給（施設数など）の回復状況を分析し、**「2019年比での回復度合い」**を可視化することを目的としています。

Google ColaboratoryとPandas/Plotlyを活用し、生データ（和暦・西暦混在、ロング形式）のクレンジングから、月次・年次での詳細な回復軌跡分析までをカバーしています。

✨ クレンジングの技術的ハイライト（DRY原則の適用）

本プロジェクトでは、分析前のデータクレンジングの品質と保守性を高めることに注力しました。

年号解析の統合:
元のデータに混在する**「和暦（平成・令和）」と「西暦（YYYY）」を統一的に処理するため、データ内容とファイル名の両方のパターンに対応できる単一のextract_year関数**に統合しました。これにより、コードの冗長性（DRY違反）を解消し、将来の年号変更にも対応しやすい構造を実現しました。

ロング形式データの月次集計:
施設カテゴリ（facility_）ごとに詳細データが複数行にわたって記録されている「ロング形式」のデータに対し、Pandasのgroupby().sum()を活用し、分析に適した**「1行＝1月・1地域」の集計データ**に変換しました。
